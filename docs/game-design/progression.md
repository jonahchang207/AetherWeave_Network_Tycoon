# Progression & Game Loop

## Phase 1 — The local startup

**Setting:** Verdant Shards

| Action | Challenge |
|--------|-----------|
| Take a loan; rent colo space | Cash flow vs. loan payments |
| Overhead Ley-Glass on light-posts | Local competitor AI undercuts prices |
| Connect neighborhoods (last-mile) | “Squirrel-golem” cable chews — basic outages |

**Win condition (phase):** Positive MRR after transit; stable local ARP; first SLA tier unlocked.

## Phase 2 — Regional dominance

| Action | Challenge |
|--------|-----------|
| Trench fiber to neighboring cities | Price wars |
| First **IXP peering** agreement | Implement **QoS** for business SLAs |
| Business-class contracts | Residential vs. enterprise contention |

## Phase 3 — Continental expansion

| Action | Challenge |
|--------|-----------|
| **DWDM** upgrades on long-haul without new trenching | Routing loops if misconfigured |
| Acquire failing ISPs | **Right-of-way** bribes at borders |
| Multi-region ARP policy | Geopolitical friction |

## Phase 4 — Global Tier-1 supremacy

| Action | Challenge |
|--------|-----------|
| **Sub-Aetheric** trans-oceanic cables | Ancient Tier-1 guild negotiations |
| Become transit provider for others | **DDoS** (Demonic Denial of Service) |
| Planet-scale redundancy | Kraken-severed cable → BGP reconvergence without collapse |

## Monetization (product)

- **Platform:** PC (Steam / GOG) — mouse & keyboard required.
- **Model:** Premium (~$29.99).
- **DLC (post-launch):**
  - *Void-Net* — LEO satellite constellations for remote shards.
  - *Cyber-Incursion* — espionage, theft, BGP hijacking depth.
